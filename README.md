# OpenMW Mod Template

A template for building OpenMW mods, designed to be used as a base for new projects to quickly get going with. Made to be used with [this modders' guide](https://modding-openmw.com/guides/modders/).

#### Installation

**OpenMW 0.48 or newer is required!**

[This page](https://modding-openmw.com/getting-started/managing-mods/) explains how to install OpenMW mods.
