Streamlined version of UI Modes, focused on streamlined menu interactions

Requires OpenMW 0.49 or newer!

Features (each feature can be separately enabled or disabled in mod settings):

  • Separate UI pages for Map, Spells, Stats, Journal, and Magic
  • Key bindings 'M' (Magic), 'Shift + M' (Map, optional), 'C' (Stats), 'I' (Inventory)
  • Use right click to exit any menu
  • Attack button (left mouse click) automatically prepares the last used weapon/spell.
  • Ability to rotate 3rd person camera in inventory mode and during dialogs (controls: left/right arrow keys, controller right stick).
  • Download the mod from Here
  • Extract the zip to a location of your choosing.
  • Paste the location you extracted it to into the form below, hit Generate Config, and paste the result into openmw.cfg.

Where would you like to install this mod?



# UI Modes Slim Data Directories
data=C:/Games/OpenMW-Mods/UI Modes Slim/.

# UI Modes Slim Content Files
content=uiModesSlim.esp

# UI Modes Slim Nested OpenMW.cfgs
# Instead of ^
config=C:/Games/OpenMW-Mods/UI Modes Slim/

Alternatively, you may simply (manually) add the following entry to your openmw.cfg: config=C:/the/location/of/ui_modes_slim

For example, if you have the ui_modes_slim folder in the same location as openmw.cfg, you can simply do this: config=ui_modes_slim

This will ensure UI Modes Slim loads after any other mods in your config chain and also apply its settings for you automagically.

You may, of course, apply these settings manually via settings.cfg, but this is more tedious and fallible:
